### Task 1: Exploring and Visualizing Iris Dataset

#### Objective

To explore and visualize the Iris dataset using Python, Pandas, Matplotlib, and Seaborn.

#### Dataset

Iris Dataset

#### Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab

#### Work Performed

* Loaded and explored the dataset
* Checked dataset shape and columns
* Generated summary statistics
* Created scatter plots
* Created histograms
* Created box plots

#### Key Findings

* Dataset contains 150 samples.
* Three flower species are present.
* Petal features help distinguish species.
* Histograms and box plots provide useful insights into feature distributions.
